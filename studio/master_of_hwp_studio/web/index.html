파일 열기
한글의 달인 사용 방법

1. 시작하기

  1. 오른쪽 위 파일 열기 를 누르고 .hwp / .hwpx 문서를 선택하세요.
  2. 문서가 열리면 상단 바에 파일명이 표시됩니다.

2. AI 에게 편집 요청하기

오른쪽 AI 작업 패널 아래쪽 입력창에 자연어로 요청하세요.

"첫 번째 문단을 공식 문서체로 바꿔줘"
"문단 5 뒤에 결론 추가해줘"
"선택한 부분을 한 줄로 요약해줘"
"보도자료를 공지사항으로 바꿔줘"

💡 미리보기 카드가 뜨면 삽입 을 눌러 실제 문서에 반영하세요. 마음에 들지 않으면 그냥 다음 요청을 입력하면 됩니다.

3. 저장하기

편집 후 상단 저장 버튼을 누르면 원본 옆에 .edited.hwpx 로 저장됩니다. 원본은 절대 덮어쓰지 않습니다.

4. 지금 지원되는 것 / 안 되는 것

기능HWPX (.hwpx)HWP (.hwp)
열기 / 읽기✅✅
문단 교체✅⚠️ 같은 길이만
표 셀 편집✅❌ (다음 버전)
문단 삽입 / 삭제❌ (다음 버전)❌ (다음 버전)

5. Claude Desktop 에서도 바로 쓰기 (선택)

터미널에서 mohwp mcp-config 를 실행하면 Claude Desktop 설정에 붙여넣을 JSON 이 출력됩니다. 설정 후 Claude Desktop 재시작 → 자연어로 HWP 편집 도구가 자동 연결됩니다.
